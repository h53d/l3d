Quick answer

Lite3D: JT File Viewer is a native Apple app for opening Siemens JT files on Mac, iPhone, iPad, and Apple Vision. It inspects product structure, renders tessellated assemblies offline with Metal or RealityKit, exports STL or GLB, and does not upload your files.

Overview

What is Lite3D: JT File Viewer?

Lite3D: JT File Viewer is a focused Apple-native viewer for Siemens JT engineering visualization files. It helps manufacturing, engineering, and review teams open tessellated JT assemblies, inspect decoded product structure, review materials and colors, and use spatial review on visionOS without CAD authoring, cloud uploads, or account sign-in.

Features

What file formats does Lite3D support?

Lite3D imports JT (.jt) files only. It is not a general CAD converter. For export, Lite3D can write tessellated model data to STL and GLB.

What can you do with Lite3D on macOS, iOS, and visionOS?

  • Assemblies: Open multi-file JT assemblies with local folder access for referenced part files
  • Structure: Browse the decoded product tree, select nodes, and read import diagnostics
  • Viewport: Use Solid, Transparent, and Wire display modes with pan, zoom, and rotate
  • Materials: View JT material and color information where the file provides it
  • Capture: Save viewport screenshots and screen recordings for review and sharing
  • Recent files: Reopen recent JT files and restore assembly-folder access on supported platforms
  • Privacy: No ads, no tracking, and no data collection. Offline by design

FAQ

What is a JT file?

A JT file (.jt) is an open 3D visualization format widely used to share product and assembly data in engineering and manufacturing workflows. Its structure and data rules are defined by ISO 14306, the international standard for the JT format, which covers product structure, tessellated geometry, and related visualization content.

What is the relationship between JT files and Siemens?

The JT format was originally developed by Siemens for lightweight 3D visualization and collaboration. It is now maintained as an open format under ISO 14306 and is widely used with Siemens products such as NX and Teamcenter, as well as by other CAD and PLM tools that publish JT visualization data. Lite3D is an independent viewer and is not affiliated with Siemens.

What is Lite3D: JT File Viewer?

Lite3D: JT File Viewer is a native Apple app for opening Siemens JT engineering visualization files on Mac, iPhone, iPad, and Apple Vision. It is a viewer, not a CAD editor.

Does Lite3D import formats other than JT?

No. Lite3D imports JT (.jt) files only. It can export tessellated model data to STL and GLB.

Can Lite3D open multi-file JT assemblies?

Yes. Lite3D can open JT assemblies that reference local part files when the referenced files are available and the app has folder access on supported platforms.

Does Lite3D work offline?

Yes. JT import, viewport review, screenshots, screen recording, and STL or GLB export run offline on your device. Lite3D does not upload your files to a server.

What platforms does Lite3D support?

Lite3D supports macOS, iOS, and visionOS. macOS and iOS use native Metal rendering, and visionOS uses RealityKit for spatial review.

What are the system requirements for Lite3D?

Lite3D requires macOS 26, iOS 26, or visionOS 26 or later.

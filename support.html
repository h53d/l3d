Lite3D: JT File Viewer

A focused Apple-native viewer for Siemens JT engineering visualization files.

Overview

Purpose

Lite3D: JT File Viewer helps you open tessellated JT assemblies, inspect product structure, review materials and colors, and use spatial review on visionOS — without CAD authoring, cloud uploads, or account sign-in.

Features

Key Features

  • Import: JT (.jt) files only — the app is not a general CAD converter
  • Assemblies: Multi-file JT assemblies with local folder access for referenced part files
  • Structure: Decoded product tree, node selection, and import diagnostics
  • Viewport: Solid, Transparent, and Wire display modes with pan, zoom, and rotate
  • Materials: JT material and color display where supported by the file
  • Export: STL and GLB from tessellated model data
  • Capture: Viewport screenshots and screen recording for review and sharing
  • Recent Files: Reopen recent JT files and restore assembly-folder access on supported platforms
  • Platforms: Native Metal rendering on macOS and iOS; RealityKit spatial review on visionOS
  • Private: No ads. No tracking. No data collection. Offline by design.
System

Technical Specifications

Requires macOS 26, iOS 26, or visionOS 26 or later.
Built with SwiftUI, native JT import, and Apple graphics frameworks (Metal / RealityKit).
Complete data privacy with zero network requirement for viewing and export.
